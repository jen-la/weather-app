# weather-app
A simple weather app that tells you what to wear.
### Tech
- HTML, CSS, JavaScript, Bootstrap
- Open Weather API
### How to run
You will need to generate your own API key from https://openweathermap.org/api and enter it where indicated in `/src/index.js`.
